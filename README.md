# Birthday Gift App

This project provides a simple environment for hosting the `1.tsx` React component using [Vite](https://vitejs.dev/).

## Development

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

## Production Build

```bash
npm run build
```

The static site will be generated in the `dist` folder. You can serve it with any static hosting service.
